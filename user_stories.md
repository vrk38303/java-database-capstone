# User Stories

## Admin
- As an admin, I want to add new doctors so that patients can book appointments.
- As an admin, I want to view all doctors so that I can manage clinic staff.
- As an admin, I want to remove doctors so that outdated records are not shown.

## Doctor
- As a doctor, I want to log in securely so that I can access my dashboard.
- As a doctor, I want to view all my patient appointments so that I can manage my schedule.
- As a doctor, I want to update appointment statuses after consultations.

## Patient
- As a patient, I want to sign up and log in so that I can book appointments.
- As a patient, I want to search doctors by specialty and availability.
- As a patient, I want to view my booked appointments.
